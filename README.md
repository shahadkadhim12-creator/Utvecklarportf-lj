## Beskrivning
Detta är en personlig portfolio byggd med HTML, CSS och JavaScript.

## Funktioner
- Responsiv navbar med hamburger-meny
- Hero-sektion med porträttbild och sociala länkar
- About Me, Skills, Portfolio, Testimonials och Kontakt-sektioner
- Formulär med validering: omöjligt att skicka felaktig input
- Flex och Grid används för layout
- DRY-princip tillämpad i CSS

## Principer och metoder
- Flexbox och CSS Grid för responsiv layout
- Semantisk HTML för tillgänglighet
- Grundläggande JS-validering för formulär
- Återanvändbara CSS-klasser (cards, grids, buttons)
